About app...

The presented sample application is intended to be used to publish and view short stories. The application supports basic CRUD operations. Most of the data has been generated to better present all available functions. The upper bar is used for navigation, some tabs will be available after logging in (login details below). The authentication process is based on JSON Web Token(JWT).
